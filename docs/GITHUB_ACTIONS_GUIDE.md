# MIU GitHub Actions
> Auto-generated CI/CD for the Micelio
> Generated by: Mistral Codestral via Relay.app

## Available workflows

### miu_daily_sync.yml
Trigger: push to main
Action: Sync README with current ecosystem state

### miu_test_tools.yml
Trigger: push to tools/
Action: Run Python doctests on all tools

### miu_validate_df.yml
Trigger: pull_request
Action: Validate D_f calculations are reproducible

## Generate new Actions

Use the GitHub Actions Generator workflow in Relay.app:
- Describe the task in natural language
- Choose trigger event
- Codestral generates production-ready YAML
- Auto-committed to .github/workflows/

## Template

All MIU actions follow this pattern:
1. Checkout repo
2. Setup Python/Node
3. Run task
4. Commit results (if applicable)
5. Notify Telegram (optional)
